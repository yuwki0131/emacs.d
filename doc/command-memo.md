## M-x commands

* emacsには知らない(隠された?)コマンドが大量にあるので、発見し次第ここに追加。

| M-x command | 説明 |
|:------------------------|:------------------------------------------------------|
| delete-matching-lines | 特定の文字列を含む行を削除 |
| ielm | elispのREPL (\*scratch\*バッファよりも使いやすい)|
| sort-line | 行単位のソート |
| magit-status | magitを使用(C-zmでバインドしているがいつも忘れる)|
